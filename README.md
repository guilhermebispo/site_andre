## Site do Prof. André Luiz Marques Serrano

Aplicação web construída com React, Vite e Tailwind CSS para apresentar o trabalho acadêmico do Prof. Dr. André Luiz Marques Serrano.

### Configuração de conteúdo
As informações públicas do site (nome, e-mail, telefone e links sociais) ficam centralizadas em `src/shared/config.ts`.

### Desenvolvimento local
```
npm install
npm run dev
```

### Build de produção
```
npm run build
```
